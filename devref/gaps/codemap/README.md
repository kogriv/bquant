# codemap — гэпы и аналитика покрытия

**Последнее обновление:** 2026-07-24

Документы этого раздела фиксируют **ограничения, расхождения дизайн↔реализация и пробелы**
инструмента `codemap/` — статического анализатора кода, который строит queryable граф пакета.

`codemap` живёт в корне репозитория рядом с `bquant/`, не входит в wheel пакета и описан в
`codemap/DESIGN.md`, `codemap/BACKLOG.md` и `devref/architecture/package_roadmap_2026-07.md`
(стадия P1 дорожной карты).
