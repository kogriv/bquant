### ‚úÖ Swing High/Low N-bar Pattern - –ü–û–î–•–û–î–ò–¢ (–ü—Ä–∏–æ—Ä–∏—Ç–µ—Ç 3)

**–°—É—Ç—å:** –û–±–æ–±—â–µ–Ω–∏–µ Pivot/Fractal —Å –Ω–∞—Å—Ç—Ä–∞–∏–≤–∞–µ–º—ã–º –æ–∫–Ω–æ–º.

**–ê–ª–≥–æ—Ä–∏—Ç–º:**
```python
def swing_high(data, left_bars=3, right_bars=3):
    """
    Swing High: high –≤ —Ç–æ—á–∫–µ i –≤—ã—à–µ –≤—Å–µ—Ö high 
    –∑–∞ left_bars –±–∞—Ä–æ–≤ —Å–ª–µ–≤–∞ –ò right_bars –±–∞—Ä–æ–≤ —Å–ø—Ä–∞–≤–∞.
    """
    swings = []
    for i in range(left_bars, len(data) - right_bars):
        is_swing = all(
            data['high'].iloc[i] > data['high'].iloc[i-j] 
            for j in range(1, left_bars+1)
        ) and all(
            data['high'].iloc[i] > data['high'].iloc[i+j] 
            for j in range(1, right_bars+1)
        )
        if is_swing:
            swings.append(i)
    return swings
```
